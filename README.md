\# 🧮 Terminal Calculator with Persistent History



A professional, interactive Command Line Interface (CLI) calculator built with Python. This project demonstrates clean code practices, error handling, and data persistence.







\## 🚀 Features



\* \*\*Robust Math Operations:\*\* Addition, Subtraction, Multiplication, and Division.

\* \*\*Data Persistence:\*\* Automatically saves your calculation history to a `.txt` file so you don't lose your data when closing the program.

\* \*\*Smart Input Validation:\*\* Custom logic to handle both dots (`.`) and commas (`,`) as decimal separators, preventing crashes from invalid user input.

\* \*\*Dynamic UX:\*\* Uses terminal clearing for a clean, app-like experience.

\* \*\*History Management:\*\* View previous calculations or clear the history (both from memory and the disk) with a single command.



\## 🛠️ Technical Skills Demonstrated



\* \*\*Modular Programming:\*\* Organized using dedicated functions for specific tasks (DRY - Don't Repeat Yourself).

\* \*\*Exception Handling:\*\* Used `try/except` blocks to manage `ValueError` and prevent the application from crashing.

\* \*\*File I/O:\*\* Reading from and writing to external text files using the `with open()` context manager.

\* \*\*Conditional Logic:\*\* Nested loops and complex `if/elif/else` structures for navigation and confirmation flows.



\## 📋 How to Run



1\.  Ensure you have \*\*Python 3.x\*\* installed on your machine.

2\.  Download the `calculator.py` file.

3\.  Open your terminal or command prompt.

4\.  Run the command:

    ```bash

    python calculator.py

    ```



\## 📂 Project Structure



\* `calculator.py`: The main application logic.

\* `calculator\\\_history.txt`: Automatically generated file where your results are stored.
